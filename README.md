# Requirements
Python 3 installed in `/usr/bin/python3` (Tested with python 3.9)

## Getting started
Script execution:
```commandline
./incremental_interquartile_mean.py
```

Running tests:
```commandline
python -m unittest test -v
```

Running the Linter (From virtual environment**):
```commandline
pip install -r requirements.txt
pycodestyle ./*.py
```
